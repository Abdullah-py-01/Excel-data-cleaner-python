# Excel Data Cleaning Automation

## What It Does
This Python script automatically cleans messy Excel files in seconds.

No more wasting hours manually cleaning data — just run the script and get a clean, ready-to-use Excel file instantly.

## Features
- Removes duplicate rows
- Removes empty rows
- Cleans and formats column names
- Exports a clean Excel file ready to use

## Tools Used
- Python
- Pandas
- OpenPyXL

## How To Run

1. Install required libraries:
pip install pandas openpyxl

2. Place your Excel file in the same folder as the script

3. Run the script:
python clean_excel.py

4. Your clean file will be saved automatically

- Businesses with messy spreadsheet data
- Anyone who wastes time manually cleaning Excel files
- Data analysts who need clean data fast
